# Landing Page Images

Bu klasör landing (ana sayfa) için kullanılan görselleri içerir.

## Gerekli Dosyalar

1. **landing-hero.png** (1920x1080 px)
   - Ana hero bölümünde gösterilecek harita görünümü ekran görüntüsü
   - Yüksek kaliteli, net ve profesyonel görünmeli

2. **screenshot-map.png** (1280x720 px)
   - Harita ve otopark listesi görünümünün ekran görüntüsü
   - Aspect ratio: 16:9

3. **screenshot-filters.png** (1280x720 px)
   - Filtreleme ve arama panelinin ekran görüntüsü
   - Aspect ratio: 16:9

4. **screenshot-stats.png** (1280x720 px)
   - İstatistikler sayfasının ekran görüntüsü
   - Aspect ratio: 16:9

## Dosya Formatı

- Format: PNG (şeffaflık destekli) veya JPG
- Optimize edilmiş dosyalar tercih edilir (küçük dosya boyutu, yüksek kalite)
- Web için optimize edilmiş görseller kullanın

## Notlar

- Tüm görseller responsive tasarıma uygun olmalıdır
- Görseller yüklendiğinde otomatik olarak boyutlandırılacaktır
- Alt text'ler erişilebilirlik için önemlidir ve HTML'de tanımlanmıştır
